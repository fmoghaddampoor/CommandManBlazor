# CommandManBlazor

![CommandManBlazor Preview](assets/app_preview_v3.png)

CommandManBlazor is a modern, web-based file manager built with Blazor Server. It provides a dual-panel interface similar to classic file managers like Norton Commander or Total Commander.

## Features

- **Dual-Panel Interface**: Manage files and directories in two independent panels.
- **Keyboard-Centric**: Use standard shortcuts for navigation and file operations.
- **File Operations**: Copy, move, delete, and create directories.
- **File Explorer Integration**: Open directories directly from Windows File Explorer with "Open with CommandMan".

## Getting Started

### Prerequisites

- .NET 10.0 SDK

### Running from Source

1. Clone the repository.
2. Navigate to the `CommandMan.Web` directory.
3. Run `dotnet run`.
4. Open your browser at `http://localhost:5000`.

### Deployment and File Explorer Integration

For instructions on how to publish the application and add the "Open with CommandMan" context menu to Windows File Explorer, please refer to the [DEPLOYMENT.md](file:///c:/CommandManBlazor/CommandManBlazor/DEPLOYMENT.md) guide.
